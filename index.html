HTML Tables

Usage of HTML tables is recommended for data which normally belongs to a grid.

It consists of following elements:

  • <table> Main element which defines what comes in the table
  • <thead> Element which groups header content in the table
  • <tr> Element which defines rows inside the table
  • <th> Element which defines a data cell inside thead row
  • <tbody> Element which groups body content in the table
  • <td> Element which defines data cells inside a row
  • <tfoot> Element which groups footer content in the table

Responsive Tables

  1. Table resizes itself according to available space.
  2. Columns and rows can be repositioned, resized, collapsed, minimized...
  3. Different techniques:
    1. Hidden Columns - Selected by User
    2. Horizontal Scroll
    3. Fixed Header
    4. Flip Scroll
    5. User Resizeable Columns
    6. Long Two Column

@media
  only screen 
  and (max-width: 760px), (min-device-width: 768px) 
  and (max-device-width: 1024px)  {

  /* Force table to not be like tables anymore */
  table, thead, tbody, th, td, tr {
    display: block;
  }

  /* Hide table headers (but not display: none;, for accessibility) */
  thead tr {
    position: absolute;
    top: -624.9375rem;
    left: -624.9375rem;
  }

  tr {
    margin: 0 0 1rem 0;
  }
  
  tr:nth-child(odd) {
    background: #ccc;
  }
  
  td {
    /* Behave  like a "row" */
    border: none;
    border-bottom: 1px solid #eee;
    position: relative;
    padding-left: 50%;
  }

  td:before {
    /* Now like a table header */
    position: absolute;
    /* Top/left values mimic padding */
    top: 0;
    left: 6px;
    width: 45%;
    padding-right: 0.625rem;
    white-space: nowrap;
  }
}
