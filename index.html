HTML 5 Tables

Usage of HTML tables is recommended for data which normally belongs to a grid.

It consists of following elements:

  • <table> Main element which defines what comes in the table
  • <tr> Element which defines rows inside the table
  • <thead> Element which groups header content in the table
  • <th> Element which defines a data cell inside thead row
  • <tbody> Element which groups body content in the table
  • <td> Element which defines data cells inside a row
  • <tfoot> Element which groups footer content in the table

Good Table Design

What to look out for:

  • Expandable areas

  • Hover action

  • Sortable

  • Highlighting color

  • Alternate row highlighting

    1. Hidden Column - Selected by user
    2. Horizontal Scroll
    3. Fixed Header
    4. Pagination
    5. Flip Scroll
    6. Resize Columns
    7. Long Two Collumn

Horizontal Scroll Code


  .rtable {
    display: inline-block;
    vertical-align: top;
    max-width: 100%;
    overflow-x: auto;
    white-space: nowrap;// optional - looks better for small cell values
	border-collapse: collapse;
    border-spacing: 0;
  }

  .rtable--flip {
    display: flex;
    overflow: hidden;
    background: none;
  }

  .rtable--flip thead {
    display: flex;
    flex-shrink: 0;
    min-width: min-content;
  }

  .rtable--flip tbody {
    display: flex;
    position: relative;
    overflow-x: auto;
    overflow-y: hidden;
  }

  .rtable--flip tr {
    display: flex;
    flex-direction: column;
    min-width: min-content;
    flex-shrink: 0;
  }

  .rtable--flip td,
  .rtable--flip th {
    display: block;
  }

Fixed Header Code

	.fixedHeader tbody {
		display: block;
		overflow: auto; //specifies what should happen if content overflows an element's box
		height: 100%;
		width: 100%;
	}

	.fixedHeader thead tr, .fixedHeader tbody tr {
		display: table;
		width: 100%;
		table-layout: fixed; //users will see the top of the table while the browser loads and renders rest of the table
	}

@media
only screen
and (max-width: 760px), (min-device-width: 768px)
and (max-device-width: 1024px)  {

	/* Force table to not be like tables anymore */
	table, thead, tbody, th, td, tr {
	display: block;
	}

	/* Hide table headers (but not display: none;, for accessibility) */
	thead tr {
	position: absolute;
	top: -9999px;
	left: -9999px;
	}

	tr {
	margin: 0 0 1rem 0;
	}

	tr:nth-child(odd) {
	background: #ccc;
	}

	td {
	/* Behave  like a "row" */
	border: none;
	border-bottom: 1px solid #eee;
	position: relative;
	padding-left: 50%;
	}

	td:before {
	/* Now like a table header */
	position: absolute;
	/* Top/left values mimic padding */
	top: 0;
	left: 6px;
	width: 45%;
	padding-right: 10px;
	white-space: nowrap;
	}

}
