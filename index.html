HTML 5 Tables

Usage of HTML tables is recommended for data which normally belongs to a grid.

It consists of following tags:

  • <'table'> - Main tag which defines what comes in the table
  • <'tr'> - Tag which defines rows inside the table
  • <'td'> - Tag which defines the data inside the table
  • <'th'> - Tag which defines a header cell in the
  • <'thead'> - Tag which groups header content in the table
  • <'tbody'> - Tag which groups body content in the table
  • <'tfoot'> - Tag which groups footer content in the table

Good Table Design

What to look out for:

  • Expandable areas

  • Hover action

  • Highlighting color

Responsive Tables

  1. Adapting web site layout with respect to the screen resolution
  2. Columns and rows can be repositioned, resized, collapsed, minimized..
  3. Different techniques:
    1. Hidden Column
    2. Horizontal Scroll
    3. Fixed Header
    4. Flip Scroll
    5. Resize Columns
    6. Sortable
